=== TITLE ===
Control Systems Analyzer — Interactive MATLAB App


=== SHORT DESCRIPTION (1 line) ===
Interactive dark-theme MATLAB app for closed-loop control system analysis: Block Diagram, Routh–Hurwitz, Root Locus, Bode, Nichols, Poles/Zeros, Step Response (ζ, ωn, h%, tr, th, ts).


=== FULL DESCRIPTION ===
Control Systems Analyzer is a single-file MATLAB App (classdef) for the analysis of linear time-invariant (LTI) control systems in the frequency and time domain.

Designed for students, educators, and engineers, the app provides a clean dark-theme interface with 8 analysis tabs that update automatically whenever input values change.

────────────────────────────────────────
WHAT'S NEW IN v0.2
────────────────────────────────────────

► Step Response tab added (8th tab)
   - Annotated step response plot for the closed-loop system T(s)
   - Automatic computation of all standard time-domain specs:
     ζ (damping ratio), ωn (natural frequency, rad/s),
     h% (percentage overshoot), tr (rise time),
     th (peak time), ts at 2% and 5% criteria
   - Dominant-pole extraction for ζ and ωn (valid for any system order)
   - Color-coded metrics display: green / orange / red by overshoot level
   - Plot annotations: peak marker, rise time line, settling time line, ±2% band

────────────────────────────────────────
FEATURES
────────────────────────────────────────

► Block Diagram
   Visual representation of the closed-loop system with Controller Gc(s),
   Plant G(s), and optional Sensor H(s) blocks. Supports both negative
   and positive feedback configurations.

► Transfer Function Results
   Computes and displays:
   - Open-loop transfer function: Gc(s)·G(s)·H(s)
   - Closed-loop transfer function: T(s)
   - Characteristic equation

► Routh–Hurwitz Stability Analysis
   Full Routh array with automatic stability verdict:
   ✓ STABLE / ⚠ MARGINAL / ✗ UNSTABLE
   with sign-change count and RHP pole count.

► Poles & Zeros Table
   All closed-loop and open-loop poles and zeros with:
   - Real and imaginary parts
   - Magnitude and angle
   - LHP / RHP / jω-axis classification

► Root Locus
   Parametric root locus plot for K ∈ [0, K_max] with
   configurable step count, RHP shading, and legend.

► Bode Diagram
   Magnitude (dB) and Phase (°) plots with:
   - Configurable frequency range (10^n rad/s)
   - Gain Margin and Phase Margin display
   - Color-coded stability indication

► Nichols Chart
   Open-loop frequency response on the Nichols plane
   with critical point (−180°, 0 dB) marker.

► Step Response  [NEW in v0.2]
   Closed-loop unit step response with full time-domain analysis:
   - ζ and ωn extracted from dominant closed-loop poles
   - h% overshoot, tr rise time, th peak time
   - ts settling time at both 2% and 5% criteria
   - Annotated plot with ±2% steady-state band

────────────────────────────────────────
USAGE
────────────────────────────────────────

>> ControlSystemsAnalyzer

Enter numerator and denominator polynomial coefficients
separated by spaces, from highest to lowest power.

Example:   1 3 2   →   s² + 3s + 2

All plots and tables update automatically on input change.
